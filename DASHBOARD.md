# Trading Claude Dashboard Guide

## Quick Start

1. **Start the server:**
   ```bash
   python serve_dashboard.py
   ```
   
2. **Open your browser:**
   - Automatically opens at `http://localhost:9110/dashboard.html`
   - Or manually navigate to that URL

3. **Load your backtest results:**
   - Drag & drop a transaction log JSON file (from `results/` folder)
   - Or click the file picker to browse

## File Format

The dashboard expects transaction log JSON files generated by the backtest. These are automatically saved to the `results/` folder with timestamps.

**Example filename:**
```
results/transactions_20251122_194232.json
```

## Tips

1. **Compare Results:** Open multiple browser tabs to compare different backtests
2. **Export Data:** Use browser dev tools to export chart data
3. **Mobile Friendly:** Dashboard works on phones and tablets
4. **No Internet Required:** All processing happens locally

## Troubleshooting

**Port already in use?**
\`\`\`bash
lsof -ti:9110 | xargs kill -9
python serve_dashboard.py
\`\`\`

**File won't load?**
- Ensure it's a valid JSON file from trading-claude
- Check browser console for errors (F12)
- Verify file size isn't too large (>100MB)

**Charts not showing?**
- Check if Chart.js CDN is accessible
- Try refreshing the page (Ctrl+R or Cmd+R)

## Customization

The dashboard is a single HTML file (`dashboard.html`) with embedded CSS and JavaScript. You can customize:

- **Colors:** Edit CSS variables in `:root`
- **Charts:** Modify Chart.js configuration
- **Metrics:** Add new calculated fields
- **Layout:** Adjust grid layouts and sections

## Keyboard Shortcuts

- **Ctrl/Cmd + R**: Refresh dashboard
- **Ctrl/Cmd + +/-**: Zoom in/out
- **F11**: Fullscreen mode
